## ۱. دامنه‌ی ممیزی (تأییدِ پوشش)

| شرط | وضعیت |
|---|---|
| هر سه اپلیکیشنِ فرانت (customer / business / company) ممیزی شد | ✅ |
| همه‌ی سفرهای کاربر (کشف، رزرو، رستوران‌دار، ادمین، خالی) ممیزی شد | ✅ |
| همه‌ی الگوهای UI (دکمه، کارت، فرم، جدول، مودال، شیت، drawer، chart...) ممیزی شد | ✅ |
| همه‌ی الگوهای UX (جستجو، فیلتر، آنبوردینگ، اعلان، بازیابیِ خطا، بازخورد...) ممیزی شد | ✅ |
| هر یافته: مشکل / چرا / بهترین‌روش / راه‌حل دارد | ✅ (در دو سند) |
| هر یافته severity دارد | ✅ (جدولِ زیر) |
| گزارشِ نهایی تولید شد | ✅ (این سند) |

**تعریفِ severity:**
- 🔴 **Critical** — مانعِ استفاده، یا نقضِ WCAG AA (ریسکِ حقوقی)، یا شکستِ سفرِ اصلی.
- 🟠 **Major** — آسیبِ جدی به کیفیت/تبدیل/ادراکِ برند؛ قابلِ دور زدن ولی مهم.
- 🟡 **Minor** — صیقل و پرداخت؛ بدونِ مانعِ عملکردی.

## ۳. خلاصه‌ی آماری

| Severity | تعداد | سهم |
|---|---|---|
| 🔴 Critical | ۱۱ | ۱۸٪ |
| 🟠 Major | ۲۸ | ۴۷٪ |
| 🟡 Minor | ۲۱ | ۳۵٪ |
| **مجموع** | **۶۰** | **۱۰۰٪** |

**سه ریشه‌ی مشترک** که اگر حل شوند بخشِ بزرگی از یافته‌ها را پوشش می‌دهند:
1. **نبودِ مقیاسِ تایپوگرافی/spacing** (محور ۲،۳) → ریشه‌ی ۱،۲۳،۳۸،۴۱ و بخشی از تراز.
   ✅ *زیرساختش در Foundation ساخته شد؛ فقط باید در صفحات اعمال شود.*
2. **emoji به‌جای آیکون** (محور ۶) → بزرگ‌ترین عاملِ حسِ «غیرپریمیوم».
3. **دسترسی‌پذیریِ ناقص** (محور ۱۸،۱۹،۴۵،۵۹) → هم ریسکِ حقوقی هم کیفیتی.
   ✅ *skip-link/live-region/focus-ring در Foundation؛ بقیه (focus-trap, aria-label) در صف.*

## ۴. آنچه در Foundation Layer قبلاً حل شد (مرجع)
سیستمِ توکنِ مشترک (تایپوگرافی/spacing/رنگ/حرکت)، کامپوننت‌های پایه (دکمه/کارت/فرم/
skeleton/empty/error)، و پایه‌ی دسترسی‌پذیری (skip-link، live-region، focus-ring،
prefers-reduced-motion). این‌ها **زیرساختِ** رفعِ یافته‌های بالا هستند؛ اعمالِ صفحه‌به‌صفحه
در فازِ پیاده‌سازی انجام می‌شود (که هنوز شروع نشده، طبقِ دستور).

## ۵. نقشه‌ی راهِ پیاده‌سازیِ پیشنهادی (فازِ بعد — هنوز اجرا نشده)
- **موجِ ۱ (Critical):** اعمالِ توکن‌ها در صفحات · جایگزینیِ emoji با SVG · focus-trap مودال ·
  aria-label + کنتراست + اندازه‌ی هدف · responsive پنل (drawer + جدول→کارت).
- **موجِ ۲ (Major):** سفرِ discover (جستجوی زنده، فیلترِ چندتایی) · جریانِ رزرو (progress+خلاصه) ·
  داشبورد (ناحیه‌ی قهرمان + chartهای SVG) · حالت‌های loading/empty/error در صفحات.
- **موجِ ۳ (Minor):** آنبوردینگ · مرکزِ اعلان · تنظیماتِ متمرکز · دارک‌مودِ پنل · صیقلِ نهایی.

# رزرونو — نسخه‌ی نمایشی (MVP Demo)

نسخه‌ی نمایشی هر سه اپ رزرونو با **دیتای نمونه‌ی ثابت** — بدون نیاز به سرور، دیتابیس یا اینترنت.
مناسب پرزنت کردن به سرمایه‌گذار.

## اپ‌ها
- **مشتری** (`customer/`) — کشف رستوران، رزرو، باشگاه مشتریان، چت، پروفایل غذایی
- **رستوران** (`business/`) — مدیریت رزرو، سالن، لیست انتظار، کارکنان، تحلیل، چت
- **شرکت** (`company/`) — مدیریت پلتفرم، رستوران‌ها، هوش تجاری، پرداخت

## اجرا (مهم)
اپ مشتری از ES Modules استفاده می‌کند، پس **باید از طریق یک وب‌سرور محلی** باز شود،
نه مستقیم از فایل‌منیجر (مرورگر ماژول‌ها را روی `file://` بلاک می‌کند).

ساده‌ترین راه — با Python:
```bash
cd demo-mvp
python3 -m http.server 8000
```
بعد در مرورگر برو به: **http://localhost:8000**

یا با Node:
```bash
npx serve demo-mvp
```

## ورود به پنل‌ها (دمو)
- **اپ مشتری:** بدون لاگین قابل استفاده است (کشف و مرور آزاد). برای بخش‌های شخصی، کد ورود دمو: **۱۲۳۴**
- **پنل رستوران و شرکت:** دکمه‌ی «🚀 ورود سریع (نسخه‌ی نمایشی)» را بزن — یا شماره‌ی دلخواه + کد **۱۲۳۴**

## نکته
همه‌ی داده‌ها نمونه و ثابت‌اند. تغییرات (رزرو جدید، پیام و...) فقط در همان جلسه‌ی مرورگر
باقی می‌مانند و به سرور نمی‌روند.
